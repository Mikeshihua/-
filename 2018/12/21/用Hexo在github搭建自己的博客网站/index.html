用Hexo在github搭建自己的博客网站

发表于 2018-12-21 |

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

本文将介绍在MacOS环境下如何快速搭建Hexo环境，创建自己的第一篇文章，以及设置主题添加评论等操作。最后会介绍如何把博客部署到Github Pages上，让其他人可以看到我们的博客。
