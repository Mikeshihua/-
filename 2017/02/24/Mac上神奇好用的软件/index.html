Mac上神奇好用的软件

发表于 2017-02-24 |

HyperDock

官网
App Store
介绍
HyperDock是一款可以让你的mac系统的任务栏浏览器图标能像win7系统那样显示当前已打开的网页略缩图，除此之外还可以平均分配画面，往左右一拉可以缺成两个平均画面，网上拉可以变成全屏幕。

Go2Shell

官网
让你在Finder中快速的在当前路径打开终端
brew cask install go2shell

Typora

官网
我用过的最舒服的MarkDown编辑器，所见即所得！
brew cask install typora

Quick Look插件

介绍
QuickLook（快速预览）让用户可以在不开启外部软件的情况下按下空格键快速预览文件。
插件列表
适合开发者的插件列表

QLColorCode

高亮预览代码brew cask install qlcolorcode

QLStephen

预览无扩展名文本文件brew cask install qlstephen

QLMarkdown

预览Markdown文件brew cask install qlmarkdown

BetterZipQL

预览zip压缩文件brew cask install betterzipql

qlImageSize

预览时显示图片的分辨率和文件大小brew cask install qlimagesize

ProvisionQL

预览ipa文件信息brew cask install provisionql
